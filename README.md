# Fix-me

## תיאור הפרויקט
פרויקט זה מכיל אתר אינטרנט שנבנה באמצעות HTML, CSS ו-JavaScript.

## מבנה הקבצים
- `index.html`: עמוד הבית של האתר.
- `style.css`: עיצוב האתר.
- `script.js`: הלוגיקה והפונקציונליות של האתר.
- `images/`: תיקייה המכילה את התמונות בשימוש באתר.

## איך להריץ?
פשוט פתח את הקובץ `index.html` בכל דפדפן אינטרנט.
